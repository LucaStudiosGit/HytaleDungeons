# Hytale Dungeons

**Kill monsters, upgrade your gear, level up!**

Hytale Dungeons is an action roguelike mod that drops you into a 
dungeon and dares you to go as deep as possible. Every floor is
different. Every run is a new challenge. Three lives stand between you and
starting over.

---

## What is this mod?

Think Minecraft Dungeons`s tower — but in Hytale. Each floor is procedurally
generated packed full of enemies. Clear them out, reach the exit,
then pick one gear upgrade from three options before you descend deeper.
Simple to learn, hard to master, impossible to put down.

---

## Features

- **Fast hack-and-slash combat** — melee weapons, bows, and a potion hotkey
- **3-life system** — die and retry the current floor; lose all three and your run ends
- **One upgrade per floor** — pick 1 of 3 gear drops (weapon, armor, or bow) each floor
- **Escalating difficulty** — deeper floors bring more mobs, tougher variants, and new enemy types

---

## How to play

1. Join a server / world with that mod installed
2. Start a run
2. Fight through every floor, kill all the monsters
3. Pick your gear upgrade
4. Repeat until you die — or win the run

---

## Installation

1. Download the save `zip` from the Files tab above
2. Place the world folder in C:\Users\user\AppData\Roaming\Hytale\UserData\Saves
3. Join the world

> Requires Hytale server build `2026.02.19-1a311a592` or newer.
